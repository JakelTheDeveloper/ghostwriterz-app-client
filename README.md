# GhostWriterz

#### *[LIVE APP](https://ghostwriterz-app.vercel.app/)*

----------------------------------------------------------------------

## User Stories

* As a user:

    * I can view demo account

    * I can Register/Login/Logout

    * I can create, update, edit, and delete lyrics

    * I can view other lyrics in the database

## Stack Used

* Javascript(React)

* HTML5

* CSS

* NodeJs(Express)

* PostgreSQL

#### Version

-1.0.0
